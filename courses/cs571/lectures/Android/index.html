Android apps!

As you know, an 'app' runs NATIVELY in its OS - no translation layer.

Android apps are .apk files. Here is a sample .apk (app), you can copy this via a USB cable to an Android phone, make it run! You can search for more apks - RUN THEM ON AN ISOLATED DEVICE!

A .apk is a glorified .jar file, which is a glorified .zip file :)

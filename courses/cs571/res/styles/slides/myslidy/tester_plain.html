Web browsers

Web browsers have been around only since 1993..

The first browser was called WorldWideWeb.

NCSA Mosaic soon followed, and after that, Netscape.

While these early browsers DISPLAYED images (in .gif and .jpg formats) and soon after, videos (in QuickTime, Flash etc.), these were NOT platforms for *creating* them..

#include <iostream>
using namespace std;

struct Node {
    int  val=0;   // data
    Node *next=NULL; // pointer 
}; // Node

int main() {
    
    Node first;
    Node *head = &first; // head is a "pointer to a Node"
    first.val = 100; 
    cout << first.val << endl;
    cout << first.next << endl; // would be 0, ie. NULL
    head->val++;
    cout << head->val << endl;
    
    Node second;
    head->next = &second; // same as: first.next = &second more text for fun fun fun
    second.val = 200; // same as head->next->val = 200;
    cout << second.val << endl;
    head->next->val++;
    cout << head->next->val << endl;
} // main()
